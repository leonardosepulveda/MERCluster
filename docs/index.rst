MERCluster - automated analysis of single-cell datasets
=======================================================

MERCluster facilitates automated analysis of one or more single-cell datasets by
implementing the core MERlin_ architecture for analyses performed on processed
datasets (i.e. a cell-by-gene matrix).

.. _MERlin: https://emanuega.github.io/MERlin/installation.html
